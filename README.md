# Sistema Bancário em Java

Este projeto consiste em um sistema bancário desenvolvido em Java, com foco na aplicação prática dos conceitos fundamentais da linguagem e da Programação Orientada a Objetos (POO).  
O sistema é executado via terminal e simula operações básicas de uma conta bancária.

## Funcionalidades

- Cadastro de cliente
- Consulta de saldo
- Realização de depósitos
- Realização de saques
- Menu interativo no terminal
- Encerramento do sistema pelo usuário

## Tecnologias Utilizadas

- Java
- VS Code
- Git
- GitHub

## Estrutura do Projeto

sistema-bancario-java/
- ├── src/
- │   ├── model/
- │   │   └── ContaBancaria.java
- │   ├── service/
- │   │   └── BancoService.java
- │   └── Main.java
- └── README.md


## Como Executar o Projeto

1. compile o projeto
javac src/Main.java

2. Execute a aplicação:
java -cp src Main
